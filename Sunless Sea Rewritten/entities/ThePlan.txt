ExoticEffects - The name of the exotic effect to execute upon exiting this event. Format is “name,parameter1,parameter2,...” The vanilla effects are as follows:
[GAME_END_EFFECT_WIN] - Ends the game with a victory—no parameters
[GAME_END_EFFECT_LOSE] - Ends the game with a loss—no parameters
[SELECT_AVATAR] - Allows you to select your character’s avatar and name—no parameters
[CHANGE_TERRAIN] - Replaces an existing terrain prefab with something different—parameters: the name of the existing TerrainInstance to replace, and the name of the prefab in the Sea/Terrain/ directory to replace it with
[MOVE_TO_PORT] - Jumps to a different port—parameters: the name of the port to jump to
[CHANGE_CURRENT_SHIP] - Updates your ship’s model—no parameters
