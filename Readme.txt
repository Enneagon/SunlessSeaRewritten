Sunless Sea: Rewritten
	by Nine (and others)
	
	
***Overview***

Sunless Sea: Rewritten is a massive undertaking to completely rewrite Sunless Sea from the ground up. Every port, every officer, every story is completely rebuilt, creating an entirely new Sunless Sea experience.


***Installation***

Back up save files:
Go to C:/Users/[YourUsername]/AppData/LocalLow/Failbetter Games/Sunless Sea/Saves/ and back up your save files.

Unzip downloaded files:
Use 7-Zip, WinRAR, or the built-in Windows extractor to unzip Sunless Sea: Rewritten and Sunless Sea: Softcoded files.

Install game files
Copy "Sunless Sea Rewritten" folder into C:/Users/[YourUsername]/AppData/LocalLow/Failbetter Games/Sunless Sea/addon/ (create folder if needed).

Install images
Copy images from "SSR Images" folder to C:/Users/[YourUsername]/AppData/LocalLow/Failbetter Games/Sunless Sea/images/sn/icons/ (create folder if needed).

Install Sunless Sea: Softcoded
Download from https://www.nexusmods.com/sunlesssea/mods/38/.
Place contents of SSSoftcoded folder into your root folder (e.g., C:/Program Files (x86)/Steam/steamapps/common/Sunless Sea/).

Install audio:
Copy "audio" folder from Sunless Sea: Rewritten download to C:/Users/[YourUsername]/AppData/LocalLow/Failbetter Games/Sunless Sea/ (create folder if needed).
NOTE: Although playing with the Sunless Sea: Rewritten soundtrack installed is the intended experience, modded audio can cause stuttering or occasional crashes. If you have persistent issues, remove the files from the "audio" folder. Perhaps consider purchasing the soundtrack separately instead? https://charlieeastwood.bandcamp.com/album/sunless-sea-rewritten-ost

Disclaimer: Adjust paths if Sunless Sea is installed on an external drive or a different location. Replace "C:/" with the correct drive letter or location and follow the same folder structure.
